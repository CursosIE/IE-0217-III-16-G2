IE-0217 - Estructuras Abstractas de Datos y Algoritmos
Proyecto - Distancia entre texto utilizando autómata de Levenshtein.
Profesor: Ricardo Román Brenes

Autores:
1-Leonardo Hernández Chacón.
2-Ericka Zúñiga Calvo.

El presente trabajo, consiste en 4 clases principales:
  1-TrieNode: Corresponde a los nodos que tiene el trie.
  2-Trie: Crea el trie. Se pueden insertar y buscar palabras dentro de él.
  3-ListWithPointer: Se utiliza para crear arreglos con punteros.
  4-Levenshtein: Es la que lleva la implementación de todas las demás clases y el algoritmo de Levenshtein.
  
  El programa recibe una palabra, un archivo y una tolerancia, y busca todas las palabras dentro del archivo que tengan una distancia entre texto menor o igual a la tolerancia.
