template<typename D, typename P>
void ListWithPointer< D, P >::insert ( D  d)
inlinevirtual

Metodo insert.

Inserta elementos en la lista. Si la lista esta vacia, se crea la celda y se almacena el dato, los punteros first y last apuntan a dicha celda. De lo contrario, se crea la celda, se almacena el dato y se asigna el puntero last a la ultima celda agregada. Se hace la conexion entre la penultima celda y la ultima, mediante el atributo de la clase Cell: next. Aumenta en uno a n.

Parámetros
ddato tipo D que se desea insertar en la lista.
Cellcelda que se crea para almacenar el dato d.

template<typename D, typename P>
void ListWithPointer< D, P >::remove ( D  d)
inlinevirtual

Metodo remove.

Remueve una celda de la lista. Se asegura que la lista no este vacia.De ser asi se crea un puntero temp tipo celda, que apunta a la celda que almacena el dato d. Si temp apunta a first, se cambia first a apuntar al siguiente valor de la lista. De lo contrario, se crea un puntero temp2 tipo celda que apunta al valor anterior a temp. Se realiza la conexion entre temp2, y la celda siguiente a temp. Se elimina temp.

Parámetros
ddato tipo D que se desea remover de la lista.
