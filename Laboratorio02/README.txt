IE-0217 - Estructuras Abstractas de Datos y Algoritmos
Laboratorio 2 - Estructuras de datos lineales.
Profesor: Ricardo Román Brenes

Autores:
1-Leonardo Hernández Chacón.
2-Ericka Zúñiga Calvo.

El presente trabajo, tiene como objetivo la implementación en C++ de las estructuras lineales de datos:
-Listas con arreglos.
-Listas enlazadas.
-Pila.
-Cola.

Esto se logró mediante la creación de la clase virtual List.h, que contiene los siguientes métodos virtuales: insert, remove, find, assign, next, prev, printList, get, getSize , sort, emptyList. Posteriormente se crean las estructuras anteriormente mencionadas, las cuales heredan de List. En la realización de dichas clases se utilizan las herramientas de plantillas, programación dirigida a objetos y herencia.
