IE-0217 - Estructuras Abstractas de Datos y Algoritmos
Laboratorio 3 - Estructuras de datos no lineales.
Profesor: Ricardo Román Brenes

Autores:
1-Leonardo Hernández Chacón.
2-Ericka Zúñiga Calvo.

El presente trabajo, tiene como objetivo la implementación en C++ de las estructuras no lineales de datos:
-Binary Search Tree.
-Graph With Pointer.
-Graph With Array.

En la realización de dichas clases se utilizan las herramientas de plantillas, programación dirigida a objetos y herencia.
